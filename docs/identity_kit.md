# Two-Line Style Note

Use Poppins for headings and Inter for body text throughout every project and page. Maintain a blue-and-white color palette with generous spacing, clean typography, and consistent layouts.

The portfolio should communicate professionalism and practical engineering skills through real projects rather than excessive animations or decorative elements.
