# Case Study 1: Finneo – AI Powered Financial Companion

## The Problem

Many beginners struggle to understand investing, risk analysis, and financial planning because most finance tools are designed for experienced users.

## What I Did

I designed and developed Finneo, an AI-powered financial companion that helps users understand investment options and financial concepts through an intuitive interface.

Technologies used:

- Next.js
- TypeScript
- REST APIs
- Artificial Intelligence
- Git & GitHub
- Vercel Deployment

I focused on building a clean user interface, integrating AI functionality, and deploying the application online.

## Outcome

- Built a fully functional web application.
- Improved my full-stack development skills.
- Learned deployment using Vercel.
- Created a project that demonstrates practical AI application development.
