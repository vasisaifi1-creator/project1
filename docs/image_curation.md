# Real Images Preferred

For every section that represents my work, I deliberately used real images.

Examples include:

- GitHub repository
- Portfolio homepage
- AI chatbot
- Resume preview
- Project screenshots
- Profile photograph

Reason:

Real evidence is stronger than generated artwork.

---

# AI Images Used

Only decorative assets use AI.

These include:

- Hero background
- Abstract textures

Reason:

Decorative graphics improve aesthetics without replacing evidence.

---

# Editing Workflow

Every selected image passed through the following process.

1. Capture original image.
2. Crop unnecessary areas.
3. Resize.
4. Compress for web.
5. Check readability.
6. Verify branding.
7. Export.
